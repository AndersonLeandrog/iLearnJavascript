Sobre o projeto

ilearnJavascript é um projeto independente desenvolvido por mim, Anderson Leandro, esse projeto faz parte de uma coletânea de projetos que possuo em meu github, trabalho com HTML, CSS e Javascript e particularmente durante o meu processo de aprendizado em javascript tive dificuldades de aprender alguns métodos da linguagem.

Foi então que surgiu a minha ideia de criar um site onde eu poderia armazenar tudo o que eu aprendi e porque não, distribuir tudo isso para o público, principalmente para aqueles que estão tendo o primeiro contato com a linguagem. Aqui você vai encontrar o básico sobre a linguagem, mas que já vai ser o suficiente para você criar os seus primeiros projetinhos!

No curso disponibilizado nesse site você vai aprender na prática, sem enrrolações, a criar, durante o curso, uma calculadora de IMC, um aplicativo de notas e um aplicativo Pomodoro para a organização de tarefas.

Nele é usado só o básico de HTML e CSS para que a nossa aplicação flue e possua um design funcional.
