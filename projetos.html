Calculadora de IMC

Durante essa jornada iremos percorrer três projetos bem simples, mas essenciais para entendermos todo o conceito apresentado em curso.

O Primeiro projeto é de uma calculadora de IMC (Indíce de Massa Corporal) esse é o projeto mais simples de todos em que iremos usar um pouco de HTML para definir o esqueleto do aplicativo, CSS para estilizar o aplicativo e por fim Javascript para fazer alguns calculos e retornar alguns valores.

    O que nós vamos usar?
  • ✔︎ HTML tags(div, p, input e button)
  • ✔︎ CSS (css e cssflex)
  • ✔︎ Javascript (variáveis, funções, getElement's e querySelector's)

acesse em: ilearnJavascript > projetos > IMC

Aplicativo todo-List

O nosso segundo projeto, um pouco mais complexo, vai exigir de nós uma lógica mais aprofundada, será um aplicativo todo-List, um aplicativo de notas para ser mais direto. Nele iremos desenvolver um sistema completo, onde será possível adicionar, editar e remover notas.

Vamos trabalhar muito em cima de alguns conceitos de inserção e remoção de elementos dentro da DOM, fique tranquilo, teremos um capítulo específico explicando o conceito de DOM.

    O que nós vamos usar?
  • ✔︎ HTML tags(div, p, input e button)
  • ✔︎ CSS (css e cssflex)
  • ✔︎ Javascript (variáveis, funções, getElement's e querySelector's)

acesse em: ilearnJavascript > projetos > todo-List

Aplicativo Pomodoro

O terceiro e mais complexo dos nossos projetos é um aplicativo pomodoro, nele vamos trabalhar com o mesmo conceito dos anteriores e principalmente, vamos aprender a como criar um contador que terá duração de vinte e cinco minutos e mais uma duração adicional de cinco minutos, nele o usuário também poderá adicionar uma nota que será removida assim que o contador for zerado, além disso teremos três funções básicas para o contador, iniciar(start), pausar(pause) e resetar(reset).

Utilizaremos tudo o que foi aprendido nos últimos dois aplicativos e teremos um capítulo adicional, onde vamos pegar todos esses projetos e postar no nosso github, assim você terá um último desafio ao terminar o curso, manter e atualizar constantemente esses projetos no seu repositório do git.

    O que nós vamos usar?
  • ✔︎ HTML tags(div, p, input e button)
  • ✔︎ CSS (css e flex)
  • ✔︎ Javascript (variáveis, funções, getElement's e querySelector's)

acesse em: ilearnJavascript > projetos > Pomodoro
